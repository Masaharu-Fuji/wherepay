# 旅行・グループ活動 費用管理アプリ

## アプリ概要

このアプリケーションは、**旅行や一日イベントなどのグループ活動における「お金の管理」と「清算」を、位置情報と紐づけて行える Web アプリ**です。

「どこで・誰が・いくら払ったのか」をその場で簡単に記録し、活動の最後に自動で割り勘を計算してくれるため、現地でのお会計から最終清算までをスムーズにします。

バックエンドには **Laravel**、フロントエンドのビルドには **Vite** を採用

---

## ユーザ向けの説明

- **その場でサッと支出登録**
  - お店や観光地などの「現在地」と紐づけて支出を記録できます。
  - 項目（例: 交通費・食事・観光・宿泊 など）と金額、支払ったメンバーを選ぶだけのシンプルな操作です。

- **位置情報と一緒に思い出を可視化**
  - どの場所でいくら使ったのか、地図やリストで振り返ることができます。
  - 「この日はどこで何をしたか」を、支出と一緒に確認できるため、旅のログとしても活用できます。

- **最後はワンタップで割り勘結果を表示**
  - 全員分の支出をもとに、「誰が誰にいくら支払えば公平か」を自動計算します。
  - 複雑な計算やメモは不要で、画面を見ながらそのまま清算できます。

- **スマホからすぐ使える Web アプリ**
  - ブラウザからアクセスするだけで利用でき、アプリのインストールは不要です。
  - スマートフォン・タブレット・PC など、複数デバイスで利用できます。

---

## 技術スタック

- **フレームワーク**
  - バックエンド: Laravel (PHP)
  - テンプレート: Twig
- **フロントエンド**
  - ビルドツール: Vite
  - 言語: JavaScript
  - スタイル: TailwindCss

`backend/` ディレクトリ配下に Laravel プロジェクトを配置し、Vite によるアセットビルドを組み合わせる構成です。

---

## 想定ユースケース

- 友人グループでの旅行・合宿の費用管理と割り勘
- 社員旅行やチームイベントでの立替精算の簡略化
- 大学ゼミ・サークル活動の現地支出管理
- イベント運営における、当日発生する小口経費の記録とレポート作成
